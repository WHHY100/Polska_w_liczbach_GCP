Informacje o projekcie

Celem projektu jest stworzenie hurtowni danych zawierającej informacje dotyczące Polski. Zbiory danych będą sukcesywnie rozbudowywane w trakcie realizacji projektu, a wszystkie dostępne informacje będzie można pobrać po kliknięciu odpowiedniej kategorii. Tabele są przechowywane w środowisku Google BigQuery, a wszystkie skrypty ETL zostały opracowane w Pythonie. Po to by nie ograniczać dostępu do danych wyłącznie do użytkowników jednej platformy, udostępniam zbiory danych w formacie CSV, co pozwala na ich import do dowolnych systemów zarządzania danymi.

Cały kod źródłowy projektu dostępny jest do wglądu po kliknięciu przycisku "GitHub".

Źródła danych:

  • API Głównego Urzędu Statystycznego
  • API Banku Danych Lokalnych
  • API Narodowego Banku Polskiego
  • API CoinGecko

Serwis zbiera ogólne dane o odwiedzajacych by analizować ruch na stronie. Zbierane informacje to: miasto, województwo, kraj i typ przeglądarki (chrome, firefox, edge itp) odwiedzajacego. Dane te nie pozwalają na identyfikację poszczególnych użytkowników.
